# centralAdmin
Plugin pour Piwigo afin de modifier (CSS) l'apparence de l'administration.

#### LE BUT :
Contraindre l'affichage de l'administration de Piwigo à rester centrée sur une seule colonne.

#### L'ADMINISTRATION

[![Screenshot-2025-12-25-at-12-42-32-Julien-Moreau-gallery-s-Administration-de-Piwigo.png](https://i.postimg.cc/R0NCVYBm/Screenshot-2025-12-25-at-12-42-32-Julien-Moreau-gallery-s-Administration-de-Piwigo.png)](https://postimg.cc/XZ66sQq1)

Les valeurs grisées ne sont pas sensés être modifiées afin de ne pas "casser" le rendu d'origine imposé par Piwigo.  
Mais vous pouvez très bien passer outre en décochant la case prévue à cet effet devant chaque champs.

#### LIMITATIONS

* Ce plugin n'est, pour l'instant, pas traduit en Anglais. Vous verriez des noms bruts à la place...
* Ce plugin est mon tout premier de cet ampleur, des bugs/erreurs peuvent persister.
* Ce n'est que de l'habillage CSS ! Le fonctionnement normal de Piwigo n'est pas affecté.
* Le bouton "Synchronisation rapide" visible depuis la page de garde est masqué. C'est un choix arbitraire. Vous retrouverez cette fonction via Le menu `Outils > Synchroniser`
